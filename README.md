# climp
CLI Musci Player

Usage
```
climp -a <file>
climp -a <file> -d
```

First usage
```
Title: "Mitt Tempel Av Stierner Og Brennende Maaner"|(7.6splat mins)
Artist: "Djevel"
Album: "Naa Skrider Natten Sort"
Year: 2022
```

Second usage
```
 Title -- "Mitt Tempel Av Stierner Og Brennende Maaner" -- 7.6splat mins
Artist -- "Djevel"
 Album -- "Naa Skrider Natten Sort"
  Year -- 2022
```

```
Usage: climp [OPTIONS] --audio <AUDIO>

Options:
  -a, --audio <AUDIO>  
  -d, --design         
  -h, --help           Print help
  -V, --version        Print version
```
